# Release 0.1.0

## Highlights
- Initial NestJS backend scaffold with modular structure for restaurants, menus, menu items, bookings, and reviews.
- Common utilities for pagination, slug handling, datetime helpers, and consistent HTTP response transformation.
- Prisma integration with the inaugural schema and generated migration for PostgreSQL.

## Infrastructure
- Docker Compose service for PostgreSQL with persisted volume and shared network.
- Project tooling configured with ESLint, Prettier, Jest, and TypeScript build targets.

## Getting Started
- Copy `.env.example` to `.env` and set database credentials when needed.
- Run `docker compose up -d` from repository root to start PostgreSQL.
- Execute `npm install` and `npm run start:dev` inside `app/backend-restaurants` to boot the API.
